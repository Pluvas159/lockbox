# react_native_lockbox

React Native wrapper for Lockbox Secure Storage SDK

## Installation


```sh
npm install react_native_lockbox
```


## Usage


```js
import { multiply } from 'react_native_lockbox';

// ...

const result = multiply(3, 7);
```
